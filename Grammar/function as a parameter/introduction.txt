"Function pointer"와 "function object"는 C++에서 함수를 매개변수로 전달하는 두 가지 주요 방법입니다.

Function Pointer:
    함수 포인터는 함수의 주소를 저장하는 변수입니다.
    함수 포인터를 사용하여 특정 함수를 가리키고 이를 다른 함수에 전달할 수 있습니다.
    함수 포인터를 선언하고 함수를 가리키도록 할당한 다음, 해당 함수 포인터를 매개변수로 전달하여 호출할 함수에 함수를 전달할 수 있습니다.

Function Object (Functor):
    함수 객체는 함수 호출을 객체로 감싼 것입니다.
    함수 객체는 클래스로 정의되며, 함수 호출 연산자(operator())를 오버로딩하여 동작합니다.
    함수 객체는 객체로서 상태를 유지하고 함수 호출에 필요한 추가적인 동작을 제공할 수 있습니다.
    함수 객체를 선언하고 이를 매개변수로 전달하여 호출할 함수에 객체를 전달할 수 있습니다.
