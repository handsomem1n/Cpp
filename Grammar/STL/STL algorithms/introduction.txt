STL algorithm <algorithm>
STL 기반의 Searching, Sorting, Counting 알고리즘들을 구현한 global functions

1. find : 컨테이너 안에서 특정한 자료를 find
2. sort : 자료들을 크기 순으로 sort
3. reverse : 자료들의 순서를 reverse
4. remove : 조건이 만족되는 자료를 remove
5. transform : 컨테이너의 요소들을 사용자가 제공하는 변환 함수에 따라서 transform

STL 알고리즘의 분류
    - 변경 알고리즘 (Mutating Algorithm):
        변경 알고리즘은 컨테이너의 내용을 수정하는 알고리즘입니다.
        원본 컨테이너의 요소들을 수정하거나 위치를 변경하는 등의 작업을 수행할 수 있습니다.
        변경 알고리즘은 주어진 컨테이너의 내부 상태를 변경하므로, 컨테이너 자체가 변경됩니다.
            초기화 알고리즘
                fill () : 지정된 범위의 모든 요소를 지정된 값으로 채운다.
                generate() : 지정된 함수의 반환값을 할당한다.
            변경 알고리즘
                for_each() : 지정된 범위의 모든 요소에 대하여 연산을 수행한다.
                transform () : 지정된 범위의 모든 요소에 대하여 함수를 적용한다.
            복사 알고리즘
                copy() : 하나의 구간을 다른 구간으로 복사한다.
                remove() : 지정된 구간에서 지정된 값을 가지는 요소들을 삭제한다.
            삭제 알고리즘
                unique() : 구간에서 중복된 요소들을 삭제한다.
            대치 알고리즘
                replace() : 지정된 구간에서 요소가 지정된 값과 일치하면 대치값으로 바꾼다.
            정렬 알고리즘
                sort() : 지정된 정렬 기준에 따라서 구간의 요소들을 정렬한다.
            분할 알고리즘
                partition() :지정된 구간의 요소들을 조건에 따라서 두개의 집합으로 나눈다.
    - 불변경 알고리즘 (Non-Mutating Algorithm):
        불변경 알고리즘은 컨테이너의 내용을 변경하지 않고 작업을 수행하는 알고리즘입니다.
        원본 컨테이너를 변경하지 않고 새로운 값을 계산하거나, 컨테이너의 내용을 순회하거나 탐색하는 등의 작업을 수행할 수 있습니다.
        불변경 알고리즘은 원본 컨테이너의 내용을 보존하며, 결과 값을 반환하거나 다른 컨테이너에 저장합니다.
            계수 알고리즘
                count() : 주어진 값과 일치하는 요소들의 개수를 센다.
                count_if() : 주어진 조건에 맞는 요소들의 개수를 센다.
            탐색 알고리즘
                search() : 주어진 값과 일치하는 첫번째 요소를 반환한다.
                search n() : 주어진 값과 일치하는 n개의 요소를 반환한다.
                find() : 주어진 값과 일치하는 첫번째 요소를 반함한다.
                find if() : 주어진 조건에 일치하는 첫번째 요소를 반환한다.
                find_end() : 주어진 조건에 일치하는 머지막 요소를 반환한다.
                binary_search() : 정렬된 컨테이너에 대하여 이진 탐색을 수행한다.
            비교 알고리즘
                equal() : 두개의 요소가 같은지 비교한다.
                mismatch() : 두개의 컨테이너를 비교하여서 일치하지 않는 첫번째 요소를 반환한다.
                lexicographical_compare() : 두개의 순차 컨테이너를 비교하여서 사전적으로 어떤 컨테어너가 작은지를 반환한다.
